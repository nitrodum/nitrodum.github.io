Physics

Tune the forces that shape the graph. Changes apply live. Drag any photo to feel the pull.

Presets

Repulsion

Charge strength

How hard nodes push each other away. Increase to spread the graph out; decrease to let nodes cluster tighter.

Links (Spring connections)

Rest distance

Target gap between connected photos, in pixels. Higher = more spread; lower = tighter grouping.

Spring strength

How strongly connected nodes pull toward each other. High values make the graph rigid.

Project clustering

Cluster gravity

Extra pull toward each project's centre. This is what makes same-project photos group tightly together.

Cluster radius

How far from the project centre nodes are pulled toward. Think of it as the target cluster size.

Damping & stability

Velocity decay

How quickly nodes slow down each step. Lower = more momentum and bounce; higher = snappy and still.

Collision padding

Extra space around each node before it collides. Prevents overlapping.

Global gravity

Weak pull toward the canvas centre so nodes don't drift too far. Set to 0 to turn off.
